Me interesa crear productos digitales funcionales, intuitivos y visualmente claros, mientras continúo fortaleciendo mis habilidades en desarrollo web, mantenimiento de sistemas, bases de datos y construcción de soluciones tecnológicas útiles para los usuarios.

> Sistema web para gestión, seguimiento y reporte de folios internos.
