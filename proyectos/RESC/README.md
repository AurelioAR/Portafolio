# RESC – Desarrollo completo

Sistema web desarrollado para la gestión de folios internos, enfocado en el registro, clasificación, seguimiento, cierre y generación de reportes de procesos. El proyecto fue construido con ASP.NET Core, Blazor Server y SQL Server, integrando funcionalidades orientadas al control de información, organización operativa y mejora del seguimiento interno - 2026.

## Rol en el Proyecto

- Diseñé y desarrollé el sistema desde cero, participando en la estructura general, lógica funcional e interfaz de usuario.
- Implementé módulos para el registro, consulta, clasificación, seguimiento y cierre de folios.
- Desarrollé la conexión con base de datos en SQL Server para el almacenamiento y control de información.
- Apliqué autenticación por roles para controlar el acceso a funcionalidades según el tipo de usuario.
- Integré notificaciones en tiempo real mediante SignalR para mejorar el seguimiento de actualizaciones dentro del sistema.
- Desarrollé un módulo de reportes exportables a Excel para facilitar el análisis y consulta de información.

## Enfoque de Desarrollo

- Arquitectura basada en servicios especializados para mantener una estructura más ordenada, modular y escalable.
- Diseño de interfaz funcional, clara y orientada al uso interno.
- Organización de flujos para facilitar el seguimiento de folios desde su recepción hasta su cierre.
- Control de roles y permisos para mejorar la seguridad y administración del sistema.
- Generación de reportes para apoyar la consulta, revisión y control de datos.
- Mejora en la organización de procesos internos mediante una solución web centralizada.
